Installation Windows

MyLang runs on Windows. Follow these steps to install it as a system command.

Step 1 — Install PyInstaller
pip install pyinstaller
Step 2 — Build the EXE
pyinstaller --onefile --clean main.py
Step 3 — Install (Admin PowerShell)
Copy-Item "dist\main.exe" "C:\Windows\System32\mylang.exe" -Force

After installation, mylang works from any terminal on your computer.

Your First Program Beginner

Create a file called hello.my and write your first MyLang program:

hello.my
say "Welcome to MyLang!"

Run it from any terminal:

Terminal
mylang hello.my
Output✓ Success
Welcome to MyLang!

Commands Reference v1.0

MyLang v1.0 includes 3 core commands:

Command Syntax Description
say say "text" or say varname Prints text or a variable to the screen
set set name = "value" Creates a variable and stores a value
ask ask var "prompt" Asks the user for input and saves it

Variables set

Use set to store any value in a variable. Then use the variable name with say.

variables.my
set name = "Ayesha"
set city = "Lahore"
say name
say city
Output✓
Ayesha
Lahore

User Input ask

Use ask to get input from the user at runtime. The first word after ask is the variable name, and the text in quotes is the prompt shown to the user.

greet.my
ask name "What is your name? "
say "Hello, "
say name
